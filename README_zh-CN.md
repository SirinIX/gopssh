# Gopssh

[TOC]

Gopssh (go parallel ssh tool) 是一款由 Go 语言编写，高效且易上手的 ssh 工具，它可以通过 ssh 连接来批量地执行命令、执行脚本、上传文件。
该项目灵感来自于 [pypssh](https://github.com/souloss/pypssh)，在这里感谢它的作者 [souloss](https://github.com/souloss)。
